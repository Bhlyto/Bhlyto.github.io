SPAS-

[view →]

Small Programming Agents System — a local multi-agent workflow runner (Ollama + Python) with a desktop GUI and per-run workspace artifacts.

  • Designed for both casual and expert users.
  • Integrates Python & Ollama (local LLM) for code generation, compilation, and execution.
  • Provides diagnostics, manages dependencies, and supports agent configuration.
  • Feature-rich GUI: workflow history, package management, environment setup.
Python Ollama LLM Agents

BHC_SCHEDULER

[view →]

Lightweight job orchestration daemon in C with REST API and optional web dashboard for submitting, tracking, and managing cluster-wide jobs.

  • Priority queue, resource-aware allocation (CPU / RAM / GPU / disk).
  • Live job streaming via Server-Sent Events (SSE).
  • Role-based user & API-key management, cross-platform daemon.
  • Cloud pool (AWS / GCP / Azure) and Wake-on-LAN support.
C REST API Distributed

[view →]

Distributed multi-fidelity simulation framework (DomainSim) with domain-specific learning and adaptive refinement.

  • Central decision core, multi-layer simulation, adaptive refinement.
  • Feedback loop for surrogate ML models and domain-specific learning.
  • Coarse-to-fine simulation pipeline with distributed orchestration.
  • Concrete experiments included (e.g. 2D heat diffusion).
